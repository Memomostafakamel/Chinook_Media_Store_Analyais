# 📊 Chinook Media Store – Data Analysis Project

## 📌 Overview  
This project presents a complete end-to-end Data Analysis workflow using the Chinook Media Store dataset.  
The analysis covers the full analytics lifecycle — starting from raw data extraction using SQL, followed by data modeling, KPI creation, and building interactive dashboards in Power BI.


<hr/>

## 🛠 Tools Used  

- SQL – Data extraction, cleaning, and transformation  
- Power BI – Data modeling and dashboard development  
- DAX – KPI creation and advanced performance metrics  
- Star Schema Modeling – Optimized relational data structure
- Figma – designed backgrounds
  
<hr/>

## 🌟 Project Highlights  

- Designed and implemented a Star Schema data model  
- Built advanced DAX measures for KPIs and performance tracking  
- Developed 6 interactive dashboard pages  
- Created 2 drill-through reports for deeper analysis  
- Analyzed sales performance, revenue trends, and customer behavior  
- Delivered comprehensive documentation and final presentation
